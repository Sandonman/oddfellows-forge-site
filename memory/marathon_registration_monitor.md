# Marathon Registration Monitor

## Goal
Monitor the Disney World Marathon registration status.

## URL
[Disney World Marathon Registration](https://www.rundisney.com/events/disneyworld/disneyworld-marathon-weekend)

## Notification Method
Send a message on Telegram when the registration status changes to open.

## Script Plan
1. Fetch the webpage content.
2. Check if registration is open (specific text or button).
3. If status changes, send a notification via Telegram with the link.
